About this Archery Project

This Archery game is currently an in progress, continually involving project. Grab your bow, and shoot the apples out of the air in this Arcade-Shooter VR game! Draw back your bow string and fire your arrows, making sure to pick up ammo to fill your quiver as you run out. You have two minutes... go!

Development

This is a personal project I started in September 2021. The original vision was to create an arcade shooter, rapidly firing your arrows within a 2 min timer to gain the highest point possible. Always having to reload your quiver to gain the max amount of shots possible. This game is developed in Unity.

This game is all original assets created by me, using Nomade, Procreate, Blender, and Maya. Implementing my own C# programming, and rigging it using the New Input System, for Oculus and the Valve Index. This project is working on the tutorial level shooting apples in an orchard, with intention to create a rail shooter expirence, putting a player in a boat to go down a river to shoot fish. This game will have updates as development progresses. So please stay tuned....
